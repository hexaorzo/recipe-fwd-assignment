# Hex's Recipe Book

A simple, static, hostel-friendly recipe website built with HTML and CSS.

## Links

- Source code: [recipe-fwd-assignment](https://github.com/hexaorzo/recipe-fwd-assignment)
- Live demo: [hexaorzo.github.io/recipe-fwd-assignment](https://hexaorzo.github.io/recipe-fwd-assignment/)

## Overview

This project showcases quick and budget-friendly recipes that are easy to cook with minimal utensils and common ingredients.

## Pages

- Home: `index.html`
- One-Pot Maggi: `one-pot-maggi.html`
- Egg Fried Rice: `egg-fried-rice.html`
- Bread Pizza: `bread-pizza.html`
- Peanut Noodle Bowl: `peanut-noodle-bowl.html`
- Mug Cake: `mug-cake.html`

## Features

- Shared navigation across all pages
- Featured recipe cards on the home page
- Individual recipe detail pages with ingredients and step-by-step instructions
- Local image assets in the `images/` folder
- Responsive layout for desktop, tablet, and phone screens
- Centralized color theme and styling in `css/styles.css`
